### 第 3 步：申请云服务（本周，1 小时，可能要等备案）

| 事项 | 说明 | 费用参考 |
|------|------|----------|
| 阿里云账号注册 | 实名认证 | 免费 |

---

### 第 4 步：部署到服务器（备案通过后，2 小时）

按 `DEPLOYMENT_CHECKLIST.md` 操作：

1. SSH 登录服务器
2. 安装 Node.js + MySQL + Nginx + PM2
3. Git clone 代码
4. 导入数据库
5. 配置 `.env`（数据库密码 + OSS 密钥）
6. PM2 启动后端
7. 配置 Nginx 反向代理
8. 配置 SSL 证书（HTTPS）

---

### 第 5 步：上传内容（服务器部署后，2 小时）

1. 通过管理后台上传 PDF / 视频 / 封面到 OSS
2. 通过 API 或管理后台更新 `content_modules` 表，填入 OSS URL
3. 编辑卷首语文字
4. 配置 `window.CONTENT_DATA` 注入

---

### 第 6 步：测试 + 加固（上线前，1 小时）

| 事项 | 说明 |
|------|------|
| 全流程测试 | 昵称 → 浏览模块 → 打卡 → 看视频 → 确认数据库有记录 |
| 修改管理后台密码 | `.env` 中改 `ADMIN_PASSWORD` |
| MySQL 自动备份 | crontab 每日备份 |
| 监控 | PM2 日志 + 阿里云云监控 |

---

### 第 7 步：持续运营（长期）

- 定期更新 PDF/视频/卷首语
- 根据反馈改进内容和功能
